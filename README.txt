            Esto es el inicio de un juego 2D

Cada fragmento del codigo está debidamente comentado sobre que realiza.

escala del personaje, velocidad de movimiento, velocidad de animacion, etc.
    Se seguira avanzando para completar el mismo y comentando cada linea de codigo para facil entendimiento.
